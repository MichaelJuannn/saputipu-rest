## Installation

Clone dulu repo nya

## install dependencies

```bash
$ npm install
```

## Create env file

```
DATABASE_URL="mysql://USERNAME:PASSWORD@HOSTNAME:PORT/DATABASE"
JWT_SECRET ="secret"
MODEL_IP ="your model ip"
```

## Running the app

```bash
npm run start:dev
```

## Database Migration

```
npx prisma migrate dev
```

## If Migration fails

```
npx prisma db push
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
